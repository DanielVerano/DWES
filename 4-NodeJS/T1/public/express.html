Express

Express es un framework para Node.js que proporciona una serie de características y herramientas para desarrollar aplicaciones web de una forma más sencilla. Podemos consultar su documentación en el siguiente enlace: https://www.npmjs.com/package/express

Como Express es un paquete de Node.js podemos instalarlo de igual forma que para instalar cualquier otro paquete: npm install express --save
