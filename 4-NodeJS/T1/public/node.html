NodeJS

Node es un “ejecutador” de documentos JavaScript.

Node permite que JavaScript se pueda ejecutar desde el lado del servidor, ya que hasta entonces solo se podía usar en los navegadores.

Las cosas más importantes a tener en cuenta sobre NodeJS son las siguientes:

  1. Gestor de paquetes NPM: npm es un gestor de paquetes de Node.js con el que la comunidad pueden crear, compartir y reutilizar módulos y pequeñas librerías. Los módulos se pueden instalar de manera local con el comando npm install 'nombre_paquete' --save. También podemos instalarlo de manera global añadiendo la opción -g.
  2. Package.json: Para inicializar nuestro proyecto, necesitamos ejecutar primero el comando npm init, que crea automáticamente el archivo package.json, donde se definen las características principales y las dependencias de nuestro proyecto con respecto a otras aplicaciones y librerías.
